# AI Experience Log

边用 AI 干活边记录，积累素材后一键出稿。

这是一个 Claude Code 的 Skill，让 AI 自动识别对话中的「有料瞬间」并记录下来。

---

## 功能

- **自动记录**：AI 识别踩坑、发现、迭代、搞笑、突破等瞬间，自动记录
- **手动补充**：说「记一笔：xxx」手动添加素材
- **一键出稿**：说「出稿」生成 1000 字左右的文章
- **自动排版**：生成公众号可用的 HTML 预览
- **封面图提示词**：输出 AI 生图提示词（适配 Gemini）

---

## 安装

### 1. 复制 Skill 文件

把 `skill/story-logger/` 文件夹复制到你的 Claude Code 项目：

```bash
cp -r skill/story-logger ~/.claude/skills/
```

### 2. 创建素材存储目录

在你想存放素材的位置创建目录：

```bash
mkdir -p your-project/drafts
```

### 3. 修改存储路径（可选）

打开 `~/.claude/skills/story-logger/SKILL.md`，把存储位置改成你自己的路径：

```markdown
## 存储位置

所有素材和产出统一存放在：
your-project/
├── drafts/
│   └── current.json    # 当前素材列表
└── [文章标题].md        # 产出的文章
```

### 4. 初始化素材文件

```bash
echo '{"topic":"","materials":[],"created":"'$(date +%Y-%m-%d)'"}' > your-project/drafts/current.json
```

## 使用

### 自动记录

正常跟 AI 干活就行。遇到这些情况会自动记录：

| 类型 | 识别信号 |
|-----|---------|
| 🐛 踩坑翻车 | 预期≠结果、报错、折腾半天 |
| 💡 意外发现 | "没想到"、"原来可以" |
| 🔄 迭代打磨 | 改了多版、从复杂到简洁 |
| 😂 搞笑时刻 | AI 抽风、神奇 bug |
| ✨ 突破时刻 | 卡了很久终于通 |
| 🎯 方法沉淀 | 可复用的技巧 |

记录时不打断对话，段落结尾会标记 `（✓ 素材+1）`

### 手动记录

```
记一笔：刚让 Claude 写了 200 行，我说太长，它改成 20 行还能跑
```

### 查看素材

```
看看素材
```

### 出稿

```
出稿
```

AI 会：
1. 分析素材，提炼主题
2. 写一篇 1000 字左右的文章
3. 生成公众号排版 HTML
4. 输出封面图提示词

## 文章风格

人设：**极客朋友** —— 硬核但说人话的技术玩家

- 开头直接切入，不废话
- 短句为主，节奏明快
- 真实感，会说"丑哭了""折腾了两小时"
- 有态度，敢下判断
- 结尾给可复制的方法
